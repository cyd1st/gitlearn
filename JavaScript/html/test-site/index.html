Linux 简介

Linux 是一个开源的操作系统。 该操作系统的内核由林纳斯·托瓦兹在1991年10月5日首次发布， 再加上用户空间的应用程序之后，就成为了 Linux 操作系统。

Linux也广泛应用在嵌入式系统上，如

  • 手机（Mobile Phone）
  • 平板电脑（Tablet）
  • 路由器（Router）
  • 电视（TV）
  • 电子游戏机

等。在移动设备上广泛使用的Android操作系统就是创建在Linux内核之上。

HTML

HTML 的主要工作之一是赋予文本结构，使浏览器能够按照开发者的意图显示 HTML 文档。

块引用:

如果一个块级内容（一个段落、多个段落、一个列表等）从其他地方被引用，你应该把它用 <blockquote>元素包裹起来表示，并且在 cite 属性里用 URL 来指向引用的资源。

CSS

CSS 是一门基于规gui则ze的语言，可以通过指定应用于网页上特定元素或元素组的样式组，来定义规则。


    h1 {
        color: red;
        font-size: 5em;
    }

分栏

你可以通过 CSS 为许多东西添加样式，CSS 由许多模块（module）构成。你可以在 MDN 上浏览这些模块的参考内容，许多模块都被组织在自己单独的文档页面。例如，我想查找一下 MDN 的 Backgrounds and Borders 模块的参考页面，来了解它是用来做什么的、它还包括什么属性、它还有什么其他特性等。在该模块中，你也可以找到指向定义了该技术规范的链接（见下文）。

在这个阶段你不必过于担心 CSS 是如何组织的 (how CSS is structured)，但是它能帮助你更好的掌握 CSS。例如，你注意到某个属性和另外一些属性的相似点，并且它们可能确实是相同的格式。

举个具体点的例子：如上文中的 Backgrounds and Borders 模块 — 你会发现 background-color 和 border-color 这两个属性在逻辑上相通。并且它也确实如此。

    const para = document.querySelector('p');

    para.onclick = function() {
        alert('噢，噢，噢，别点我了。');
    }

onclick是一个通用回调属性。

在上述的 JavaScript 示例中，para 表示一个段落元素。
