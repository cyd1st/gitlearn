Linux 简介

Linux 是一个开源的操作系统。 该操作系统的内核由林纳斯·托瓦兹在1991年10月5日首次发布， 再加上用户空间的应用程序之后，就成为了 Linux 操作系统。

Linux也广泛应用在嵌入式系统上，如

  • 手机（Mobile Phone）
  • 平板电脑（Tablet）
  • 路由器（Router）
  • 电视（TV）
  • 电子游戏机

等。在移动设备上广泛使用的Android操作系统就是创建在Linux内核之上。
